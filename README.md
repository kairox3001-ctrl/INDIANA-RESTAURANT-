# Indiana Veg Restaurant and Cake Boutique Website

A fully functional, responsive restaurant website for Indiana Veg Restaurant and Cake Boutique, Aurangabad.

## 🌟 Features

- **Responsive Design**: Works perfectly on mobile, tablet, and desktop devices
- **Modern UI**: Clean and professional design with smooth animations
- **Navigation**: Sticky navbar with mobile-friendly hamburger menu
- **Hero Section**: Eye-catching banner with call-to-action buttons
- **About Section**: Restaurant information with key statistics
- **Menu Section**: Filterable menu with categories (Breakfast, Indian, Chinese, Fast Food, South Indian)
- **Cakes & Bakery**: Showcase of premium cakes and pastries
- **Reviews Section**: Customer testimonials with rating breakdown
- **Order Integration**: Direct links to Swiggy and Zomato
- **Contact Section**: Complete contact information, contact form, and embedded Google Map
- **Footer**: Social media links and additional information

## 🚀 Getting Started

### Prerequisites
- A web browser (Chrome, Firefox, Safari, Edge)
- A text editor (VS Code, Sublime Text, etc.)
- Git (for cloning the repository)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/kairox3001-ctrl/INDIANA-RESTAURANT-.git
cd INDIANA-RESTAURANT-
```

2. Open `index.html` in your web browser

### Deployment

#### Option 1: GitHub Pages (Free)
1. Go to your repository settings
2. Navigate to Pages section
3. Select main branch as source
4. Your site will be live at: `https://kairox3001-ctrl.github.io/INDIANA-RESTAURANT-/`

#### Option 2: Netlify (Free with optional pro features)
1. Connect your GitHub repository to Netlify
2. Deploy automatically on every push

#### Option 3: Vercel (Free)
1. Connect your GitHub repository to Vercel
2. Deploy in one click

## 🎨 Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #d4691f;      /* Main brand color */
    --secondary-color: #8B4513;    /* Secondary color */
    --accent-color: #2ecc71;       /* Accent/CTA color */
    --dark-color: #2c3e50;         /* Dark background */
    --light-color: #ecf0f1;        /* Light background */
}
```

### Content
Edit the HTML in `index.html` to update:
- Restaurant name and description
- Menu items and prices
- Cake offerings
- Contact information
- Links to Swiggy/Zomato

### Functionality
Modify `script.js` to add:
- Additional animations
- Form validation
- Backend integration
- Analytics tracking
